# RoboSentinel
This robot is mainly made for the security of our house. We can monitor our house from outside using a camera in the robot with a webserver. Interesting thing is that we created the webserver by using web development coding. We have used two micro-controllers in this robot and they are Arduino Uno and ESP32. There is also some sensors in the robot. They are gas sensor ,humidity sensor, temperature sensor and motion sensor. Gas sensor will notify us whether if there happened a sudden accident of leakage in our house by measuring the gas level in the air. Humidity and temperature sensor will measure the humidity and temperature of our house. We can see the measures from an external webserver form anywhere in the world. The motion sensor will detect if there happen a motion of something like if a thief enter our house, then it will notify us.
To make this, you need
1.ESP32
2.ESP32 Cam
3.Car kit
4.DHT11
5.MQ2
6.Lithium Ion Batteriees(3)
7.Battery case
You can control the robot from outside of your house using only your mobile. Also you can monitor the live footage as well as sensor's data.
